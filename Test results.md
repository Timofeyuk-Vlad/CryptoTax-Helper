# Тестирование CryptoTax Helper

## Функциональное тестирование

### Аутентификация и безопасность

| Cценарий | Действие | Ожидаемый результат | Фактический результат | Оценка |
|:---|:---|:---|:---|:---|
| 001-1: Регистрация нового пользователя | Ввести email, пароль, подтверждение пароля → нажать "Зарегистрироваться" | Создание учетной записи, установка FREE тарифа | Учетная запись создана, тариф FREE установлен | Тест пройден |
| 001-2: Регистрация с уже существующим email | Ввести уже зарегистрированный email | Ошибка "Пользователь с email уже существует" | Ошибка отображается | Тест пройден |
| 002-1: Вход зарегистрированного пользователя | Ввести корректные email и пароль → нажать "Войти" | Успешная аутентификация, получение JWT токена | Авторизация успешна, токен получен | Тест пройден |
| 002-2: Вход с неверными данными | Ввести неверный пароль | Ошибка "Неверный email или пароль" | Ошибка отображается | Тест пройден |
| 003-1: Настройка 2FA | В настройках безопасности нажать "Настроить 2FA" | Генерация QR-кода и секрета для Google Authenticator | QR-код и секрет сгенерированы | Тест пройден |
| 003-2: Верификация 2FA | Ввести код из приложения аутентификатора | Успешная активация 2FA, статус "ENABLED" | 2FA успешно активирован | Тест пройден |
| 004-1: Валидация API ключей Binance | Ввести корректные API ключи Binance | Успешная валидация, статус "API ключи валидны" | Валидация успешна (в демо-режиме) | Тест пройден |
| 005-1: Валидация с неверными ключами | Ввести неверные API ключи | Ошибка "Неверные API ключи" | Ошибка отображается | Тест пройден |
| 006-1: Детальный FIFO расчет | Запросить детальный FIFO отчет | Детализация по активам, операции покупки/продажи | Детальный отчет с операциями отображен | Тест пройден |
| 007-1: Генерация PDF отчета | Нажать "Сгенерировать PDF отчет" | Создание PDF файла с налоговой сводкой | PDF файл создан и доступен для скачивания | Тест пройден |
| 007-2: Генерация Excel отчета | Нажать "Сгенерировать Excel отчет" | Создание Excel файла с детализацией операций | Excel файл создан с транзакциями | Тест пройден |
| 008-1: Просмотр дашборда | Перейти на главную страницу | Отображение портфеля, лимитов, последних транзакций | Дашборд отображает все разделы | Тест пройден |
